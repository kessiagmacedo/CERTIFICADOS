# Certificates – Data Science & Computing

This repository is a collection of certificates obtained from courses related to **Data Science**, **Programming**, and **Computing**.

The objective of this repository is to organize and document the continuous learning process in areas such as:

- Data Science
- Data Analysis
- Machine Learning
- Statistics
- Programming
- Python
- Databases
- Artificial Intelligence
- Computer Science fundamentals
- Data-related tools and technologies

## Structure

Certificates are added as they are completed.  
The repository is continuously updated as new courses are finished.

Each certificate represents completed training from educational platforms, universities, or professional programs focused on technical development in data and computing fields.

## Purpose

- Document academic and technical progress  
- Maintain a centralized record of completed courses  
- Provide a transparent archive of certifications related to data and computing

## Status

This repository is **in progress** and will be expanded over time as new certifications are obtained.
